# Week 0: Introduction to Web Development 🌐

Welcome to Week 0 of the *Introduction to Web Development* course! In this session, we will cover the foundational concepts and tools that every web developer should know to get started in the world of web development.

## Overview of Web Development, Tools, and Technologies

### What is Web Development? 💻
**Web Development** refers to the process of building, designing, and maintaining websites and web applications. It involves two main areas:
- **Frontend (Client-Side)**: The part of the website that users interact with directly (e.g., layout, text, images).
- **Backend (Server-Side)**: The server, databases, and business logic that process requests and send data to the frontend.
- **Full Stack**: A combination of both frontend and backend development.

### Why Learn Web Development? 🚀
- **High Demand**: Web developers are in high demand globally. Employment in this field is expected to grow 16% between 2022 and 2032 (Bureau of Labor Statistics).
- **Creative Expression**: Build dynamic, interactive websites.
- **Job Opportunities**: Work for companies, freelance, or start your own business.

### Key Technologies in Web Development 🛠️
- **Frontend**: 
   - **HTML**: HyperText Markup Language (structure).
   - **CSS**: Cascading Style Sheets (presentation).
   - **JavaScript**: Adds interactivity and functionality.
- **Backend**: 
   - **Databases** (e.g., MySQL, MongoDB).
   - **Server-side Languages** (e.g., Node.js, PHP, Python).

---

## The Three Core Technologies of Web Development ⚙️
- **HTML** (Structure)
- **CSS** (Style)
- **JavaScript** (Behavior)

### Overview of HTML 📝
HTML defines the structure and content of web pages. It uses elements and tags like `<div>`, `<p>`, `<a>`, and `<img>` to build content.

Example:
```html
<h1>Hello, World!</h1>
```

### What is CSS? 🎨
CSS controls the look and feel of a webpage by applying styles to HTML elements. 

Example:
```css
h1 { color: blue; }
```

### What is JavaScript? ⚡
JavaScript enables dynamic interactivity and behavior on web pages.

Example:
```javascript
document.getElementById('under-button').innerHTML = 'Hello World';
```

---

## The Client-Server Model 🌐
- **Client**: The browser or app where users interact with the webpage (e.g., Chrome, Safari).
- **Server**: A remote machine that processes requests and serves data.
- **HTTP**: HyperText Transfer Protocol is the communication method used between clients and servers.

## Git and GitHub Basics 🗂️
### Git Commands:
- **git init**: Initialize a Git repository.
- **git add .**: Stage all changes for commit.
- **git commit -m "message"**: Commit changes with a message.
- **git push**: Push changes to the remote repository on GitHub.

### GitHub Setup:
- Sign up for a GitHub account [here](https://github.com).
